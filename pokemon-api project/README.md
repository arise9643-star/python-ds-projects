# Pokémon Info Finder 🎮

A Python script that fetches real Pokémon data from the PokéAPI.

## What it does
- Search any Pokémon by name
- Shows ID, height, weight and abilities
- Handles errors cleanly

## How to run it

1. Install the required library:
pip install requests

2. Run the script:
python api.py

3. Type any Pokémon name when asked

## Example Output
Name : Pikachu
ID : 35
Height : 4
Weight : 60
Abilities:
 - static
 - lightning-rod

## Built with
- Python
- Requests library
- PokéAPI (https://pokeapi.co)
